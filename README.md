# Premium Stock Analyzer (국내/해외 주식 분석기)

이 앱은 국내(KOSPI, KOSDAQ) 및 해외(S&P 500, NASDAQ 등) 주요 시장 지수, 개별 종목의 가격 정보, 차트 분석, 재무 지표, 관심 종목 및 포트폴리오 관리 기능을 갖춘 프리미엄 다크 모드 대시보드입니다.

## 주요 기능

1. **실시간 시장 지수**: 주요 시장 지수의 실시간(지연 반영) 데이터 모니터링
2. **기술 분석 차트**: TradingView Lightweight Charts를 이용해 봉차트 및 이동평균선(MA5, MA20, MA60, MA120)을 렌더링하며 마우스 휠 줌/드래그 지원
3. **상세 재무 메트릭**: PER, PBR, 배당수익률, 시가총액, 52주 최고/최저가 등
4. **뉴스 및 AI 감성 분석**: 검색된 주식 맞춤형 뉴스 기사 생성 및 심리 점수(호재/악재) 시뮬레이션
5. **동적 주식 비교**: 2개 종목을 검색하여 1년 기준 상대 수익률 변동 차트 및 핵심 지표 병렬 비교
6. **관심 종목(Watchlist) & 포트폴리오**: 종목 추가/제거 및 평단가 수량 입력을 통한 수익률/평가액 실시간 계산 (`localStorage` 연동)

## 프로젝트 구조

```text
stock-analyzer/
├── backend/
│   ├── main.py          # FastAPI 애플리케이션 및 주식 데이터 API 엔드포인트
│   └── requirements.txt # Python 종속성 파일 (FastAPI, yfinance 등)
├── frontend/
│   ├── index.html       # 대시보드 마크업 구조
│   ├── style.css        # 프리미엄 다크 모드/글래스모피즘 디자인 시스템
│   └── app.js           # 프론트엔드 차트 바인딩, localStorage 동기화, UI 제어
├── run.sh               # 간편한 가상환경 셋업 및 실행 쉘 스크립트
└── README.md            # 안내서
```

## 시작하기

프로젝트 루트 디렉토리에서 터미널을 열고 시작 스크립트를 실행하면 가상환경 설정, 패키지 설치 및 서버 구동이 한 번에 진행됩니다:

```bash
./run.sh
```
